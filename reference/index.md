# Package index

## All functions

  : Validate LAS file names according to the ADV standard

  : Get the Coordinate Reference System (CRS) of LAS files

  : Get data in a standardized format and structure

  : Get acquisition date from LAS files
- [`get_density()`](https://wiesehahn.github.io/managelidar/reference/get_density.md)
  : Get approximate point and pulse density of LAS files
- [`get_extent()`](https://wiesehahn.github.io/managelidar/reference/get_extent.md)
  : Get the spatial extent of LAS files
- [`get_header()`](https://wiesehahn.github.io/managelidar/reference/get_header.md)
  : Retrieve LAS file headers (metadata)
- [`get_intersection()`](https://wiesehahn.github.io/managelidar/reference/get_intersection.md)
  : Get intersecting LAS files
- [`get_lasversion()`](https://wiesehahn.github.io/managelidar/reference/get_lasversion.md)
  : Get the LAS version of point cloud files
- [`get_names()`](https://wiesehahn.github.io/managelidar/reference/get_names.md)
  : Get LAS file names
- [`get_penetration()`](https://wiesehahn.github.io/managelidar/reference/get_penetration.md)
  : Compute pulse penetration ratios from LAS headers
- [`get_summary()`](https://wiesehahn.github.io/managelidar/reference/get_summary.md)
  : Compute summary metrics for individual LAS files and optionally save
  as JSON
- [`is_classified()`](https://wiesehahn.github.io/managelidar/reference/is_classified.md)
  : Check whether LAS files are classified
- [`laz_files`](https://wiesehahn.github.io/managelidar/reference/laz_files.md)
  : Laz files
- [`plot_extent()`](https://wiesehahn.github.io/managelidar/reference/plot_extent.md)
  : Plot the spatial extent of LAS files
- [`print_pulseinfo()`](https://wiesehahn.github.io/managelidar/reference/print_pulseinfo.md)
  : Print summary information about pulse density and penetration ratio
- [`set_names()`](https://wiesehahn.github.io/managelidar/reference/set_names.md)
  : Set LAS file names according to ADV standard
