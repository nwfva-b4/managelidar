# Build links for a STAC item

Build links for a STAC item

## Usage

``` r
build_item_links(
  item_id,
  items_dir,
  collection_dir,
  root_path,
  root_title,
  collection_title
)
```

## Arguments

- item_id:

  Item ID

- items_dir:

  Path to items directory

- collection_dir:

  Path to collection directory

- root_path:

  Absolute path to root catalog

- root_title:

  Title of the root catalog

- collection_title:

  Title of the owning collection

## Value

List of link objects
