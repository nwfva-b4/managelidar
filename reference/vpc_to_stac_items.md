# Convert VPC features to STAC items

Convert VPC features to STAC items

## Usage

``` r
vpc_to_stac_items(vpc_obj, collection_dir, items_dir)
```

## Arguments

- vpc_obj:

  VPC object (list with \$features)

- collection_dir:

  Path to collection directory

- items_dir:

  Path to items directory

## Value

List of STAC item objects
